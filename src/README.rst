************
Installation
************

Requirements
============

- Python 3.*
- pip 20.0.2

Installing by using PyPi
========================

Install
-------
.. code-block:: bash

	git clone https://github.com/Intelligent-Systems-Phystech/2023-Project-139.git /tmp/2023-Project-139
	python3 -m pip install /tmp/2023-Project-139/src/

Uninstall
---------
.. code-block:: bash

  python3 -m pip uninstall mylib
